## About the Project
This project displays a list of products with their prices and categories such as electronics and clothing.  
When a user selects a category from the dropdown menu, only the relevant products are displayed.  
The filtering happens dynamically without reloading the page.

---

## How the Project Works
1. Products are written directly in HTML  
2. Each product belongs to a category  
3. A dropdown menu is used to select the category  
4. JavaScript shows or hides products based on selection
